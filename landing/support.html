Need help with Code Awake? Here are the basics and where to reach the developer.

About the App

Code Awake is a macOS menu bar utility that keeps your Mac available for builds, downloads, presentations, remote sessions, and supported closed-lid workflows.

Quick Checks

  • Make sure Code Awake is running in the macOS menu bar.
  • Enable Keep Mac Awake before starting long-running work.
  • Use Launch at Login if you want Code Awake ready after every restart.
  • Closed-lid behavior can depend on macOS, power, display, and hardware conditions.

Safe Use

When using awake or closed-lid workflows, make sure your Mac has suitable ventilation, power, and hardware support for your setup. You remain responsible for how your device is used and monitored during long-running or unattended activity.
